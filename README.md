* This is my personal repository for a small assortment of python programs.  
** PDF Merger:
 - Takes input of one or more PDFs and combines them to a single PDF.
 - To use download .exe in the latest version Dist folder. 
  
** Image Background Remover:
  - Removes the background of an image based on pixel match
  - No advanced removal tools

** Image Converter:
  - Converts images between a variety of types (PNG, JPEG, JPG, GIF, ICO, ETC)
  - Use image_conv_cmd.py formats to list all formats 
  - Needs Pillow python library

** Apple Image Converter:
  - Converts apple photo directories to standard formats (JPG and MP4)
  - Copies folder heierarchy and transfers alternative files (PNG, JPEG, etc)
